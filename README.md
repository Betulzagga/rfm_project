# rfm_project
# Customer Segmentation using RFM Analysis

This project performs customer segmentation using the RFM (Recency, Frequency, Monetary) model.

## Project Goal

Identify valuable customers and segment them into meaningful groups to support marketing strategies.

## Dataset

Online Retail Dataset containing transactional data from an e-commerce platform.

## Methodology

RFM model calculates:

Recency → How recently a customer purchased  
Frequency → How often they purchase  
Monetary → How much money they spend

Customers are then segmented based on these metrics.

## Tech Stack

Python  
Pandas  
NumPy  
Matplotlib  
Seaborn

## Output

Customer segments such as:

Champions  
Loyal Customers  
Potential Loyalists  
At Risk  
Hibernating

## Example Segmentation

| Customer Segment | Description |
|---|---|
| Champions | High value, frequent buyers |
| Loyal Customers | Regular buyers |
| At Risk | Previously active but declining |
| Hibernating | Low engagement |

## Business Impact

RFM segmentation helps companies:

• Personalize marketing campaigns  
• Increase customer retention  
• Identify high-value customers
